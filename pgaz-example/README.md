# Azure Example PostgreSQL Database

## How To build
**Make sure you have Docker Running**
```
1. Run the following command depending on which environment you want to build:
```
make dev
-or-
make prod
```
2. Follow the on-screen prompt.
  - Copy and paste the given URL into a local browser session (This sometimes fails, just retry with the same URL).
```
WARNING: To sign in, use a web browser to open the page https://microsoft.com/devicelogin and enter the code ######### to authenticate.
```
  - Copy and paste the given code into the Microsoft Login field and continue.
  - Select your username and continue.
